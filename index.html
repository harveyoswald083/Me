About Me

Mobile Developer & Cloud-Native Solutions Architect

I'm a Mobile Developer specializing in native iOS and Android development with cloud-integrated solutions. I build high-performance mobile applications that leverage cloud infrastructure to deliver scalable, secure, and feature-rich experiences across platforms.

With expertise in Swift, Kotlin, React Native, and cloud services (Firebase, AWS Mobile SDK, Azure Mobile Apps, Google Cloud), I create mobile applications that seamlessly integrate with cloud backends for authentication, real-time synchronization, push notifications, analytics, and serverless functions. I'm passionate about building native mobile apps with clean architecture patterns, implementing robust state management, integrating RESTful and GraphQL APIs, and deploying to App Store and Google Play. My work focuses on creating intuitive UIs, optimizing performance and battery life, implementing offline-first strategies, and ensuring secure cloud connectivity with proper authentication and data encryption.
